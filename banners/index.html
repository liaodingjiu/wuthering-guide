🎰 How Banners Work

Wuthering Waves has four concurrent banner types. Here's what each one does and which to pull on:

  • Limited Character Banner (Convene): the main banner everyone pulls on. Features one limited 5★ character with three 4★ rate-ups. 80 pulls to hard pity, 50/50 system. This is where you should spend your Astrite.
  • Limited Weapon Banner: features the signature weapon of the current limited character. No 50/50 — when you hit pity, you are guaranteed the featured weapon. 80 pity cap. Only pull here if you really love the character and can afford it.
  • Standard Banner (Utterance of Marvels): permanent banner with all standard 5★ characters and weapons. Use your free Lustrous Tides here — never spend Astrite on this banner.
  • Beginner Banner (Novice Convene): discounted 40-pull banner for new accounts only. Guarantees a random standard 5★ within 40 pulls. Pull this first, then switch to limited banners.
💡 Pull strategy: Beginner Banner (40 pulls) → Limited Character Banner (save 160 pulls for guarantee) → Limited Weapon Banner (optional, characters first). Never spend Astrite on Standard. Use the Tier List → to decide who's worth pulling.

❓ Banner FAQ

How long does each banner last?
Each phase lasts approximately 3 weeks (21 days). A full patch has two phases, so each version runs about 6 weeks total. New banners start immediately after the previous one ends — there is no downtime between phases.

What's the difference between Phase 1 and Phase 2?
Phase 1 features the brand-new 5★ character for that patch, plus their signature weapon. Phase 2 typically features a rerun character (or a second new character in special patches). 4★ rate-ups also change between phases.

Does pulling on a rerun banner share pity with the new character banner?
Yes — all limited character banners share the same pity counter. Your pulls on Jinhsi's rerun banner count toward pity on the next new character banner, and vice versa.

When do we know the next banner?
Kuro Games typically announces the next version's characters through "drip marketing" on social media about 1-2 weeks before the current version ends. The official Special Program livestream confirms banner order roughly 1 week before the patch drops.

Are 4-star characters guaranteed on limited banners?
Every 10 pulls guarantees at least one 4★ item — either one of the three featured rate-up 4★ characters, a 4★ weapon, or an off-banner 4★. You are guaranteed to get one of the featured 4★ rate-ups every other 4★ pull (50% rate-up, then guaranteed).
