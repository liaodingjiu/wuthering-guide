📋 How to Redeem Codes

In-Game Redemption

1Open Wuthering Waves and tap the Settings icon (⚙️ gear) in the top-right menu
2Go to Account → Redeem Code
3Enter the code exactly — codes are case-sensitive
4Tap Confirm to submit
5Rewards are sent to your in-game Mailbox

Web Redemption

Redeem online at wutheringwaves.kurogames.com/en/gift. You will need to log in with your Kuro Games account and select your server and character.

❓ Frequently Asked Questions

How often are new codes released?
New codes are typically released during version updates (e.g., 2.0 → 2.1), milestone celebrations (player count, anniversaries), holidays, official livestreams, and server maintenance compensation. Check this page weekly — we update within 24 hours of a new code drop.
Why is my code not working?
Common reasons: (1) The code has expired — most codes have a limited redemption window. (2) Typo — codes are case-sensitive; use the Copy button instead of typing. (3) Server/region mismatch — some codes only work on specific servers. (4) Already redeemed — each code can only be used once per account. (5) Redemption cap reached — some codes have a total usage limit.
Do codes expire?
Yes. Some codes have a publicly announced expiration date; others expire silently. Version-launch codes typically last 2–4 weeks. Always redeem codes as soon as possible. Our confidence badges help you gauge which codes are still likely to work.
Can I use codes on any server?
Not always. Codes from official global channels (Twitter/X, global Discord) usually work on SEA and Global servers. Codes from CN-specific events or Bilibili are often CN-only. We label region restrictions on each code card when known.
How do I know if a code is real?
Every code on this page includes a source link when available — click it to see the original announcement from Kuro Games. Codes without a verified source are clearly marked with a ⚠️ Unverified badge. We do not list obviously fake codes.
Where can I find new codes?
The best sources: Twitter/X (@WutheringWaves), the official Discord, the Wuthering Waves website News section, version livestreams, and this page. We monitor all official channels and add new codes within 24 hours. Bookmark this page (Ctrl+D) for the fastest access.
